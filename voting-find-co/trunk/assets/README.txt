Introduction:

A simple voting plugin.
Plugin is developed for users who want to add voting functionality to their articles.
To ensure that each visitor can vote only once IP address check is implemented.

Prerequisites:

WordPress 6.1.1 >=

Installation:

To install this plugin upload it's zip file on Add Plugin page and activate it once it's installed.

Configuration:

Once the plugin is activated, voting content is added at the end of each article. Simple as that.
